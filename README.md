# React Minesweeper

A React version of the Minesweeper game.

## Demo

![](react-minesweeper-demo.gif)

## Development

```
$ git clone https://github.com/shahardav/react-minesweeper.git
$ cd react-minesweeper
$ npm install
$ npm start
```

## Running the tests

```
$ cd react-minesweeper
$ npm test
```
